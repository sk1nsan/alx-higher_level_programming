The 1-my_list module
======================

Using print_sorted
-------------------
>>> MyList = __import__("1-my_list").MyList

works with integers

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

works with non integers

>>> my_list3 = MyList()
>>> my_list3.append('b')
>>> my_list3.append('e')
>>> my_list3.append('a')
>>> my_list3.append('c')
>>> my_list3.append('d')
>>> my_list3.print_sorted()
['a', 'b', 'c', 'd', 'e']
>>> print(my_list3)
