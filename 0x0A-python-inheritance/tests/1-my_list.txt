The 1-my_list module
======================

Using print_sorted
-------------------
>>> MyList = __import__("1-my_list").MyList

works with integers

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

no errors with empty list

>>> my_list2 = MyList()
>>> my_list2.print_sorted()
[]
>>> print(my_list2)
[]
